⚙️ Configuração Google Sheets

📋 Como configurar o Google Sheets

Clique para ver instruções
  1. Crie uma nova planilha no Google Sheets
  2. Na planilha, vá em Extensões → Apps Script
  3. Apague o código padrão e cole o código fornecido na seção abaixo
  4. Salve o projeto (Ctrl+S)
  5. Clique em Implantar → Nova implantação
  6. Em "Tipo", selecione "Aplicativo da Web"
  7. Em "Quem tem acesso", escolha "Qualquer pessoa"
  8. Clique em "Implantar"
  9. Copie a URL do Web App (não a URL da planilha) e cole no campo acima
  10. Clique em "Testar Conexão" para verificar se está funcionando

⚠️ Importante: Use apenas a URL do Web App que termina com /exec, não a URL da planilha do Google Sheets.

📋 Como configurar (Passo a Passo)

  1. Crie uma nova planilha no Google Sheets
  2. Vá em Extensões → Apps Script
  3. Cole o código fornecido acima e salve
  4. Vá em Implantar → Nova implantação
  5. Escolha "Aplicativo da Web", acesso "Qualquer pessoa"
  6. Copie a URL do Web App gerada aqui

⚠️ Importante

⚠️ Importante: Após colar o código no Apps Script, vá em Implantar → Nova implantação, escolha "Aplicativo da Web", defina acesso para "Qualquer pessoa" e copie a URL do Web App (não da planilha).

🔧 Solução de Problemas

1. Erro de permissão/autorização:
  • Abra o Apps Script e execute a função testFunction manualmente
  • Autorize todas as permissões solicitadas
  • Aguarde alguns minutos e tente novamente
2. URL não funciona:
  • Use APENAS a URL do Web App (termina com /exec)
  • NÃO use a URL da planilha (sheets.google.com)
  • Exemplo correto: https://script.google.com/macros/s/SEU_ID/exec
3. Erro 403/Acesso negado:
  • Vá em Implantar → Gerenciar implantações
  • Clique no ícone de edição (lápis)
  • Defina "Quem tem acesso" como "Qualquer pessoa"
  • Clique em "Implantar" e use a nova URL
4. Erro de rede/timeout:
  • Verifique sua conexão com a internet
  • Tente acessar a URL diretamente no navegador
  • Se não funcionar, refaça a implantação
